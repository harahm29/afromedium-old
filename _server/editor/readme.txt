1. фон - для редактора картинка фона должна быть на 4 рх меньше по ширине и на 32 по высоте
а для въювера на 4 по ширине и 4 по высоте
2. Надпись Loading please wait еще не убнана, так как в момент загрузки мы запрещаем пользователю работать инструментами, и когда он пытается что-то клацать, эта надпись ему поясняет в чем дело, ну просто чтоб человек не мучался вопросом "а почему это тут ничего не шивелится, вчера-ж я тут работал?"
3. И ЭТО САМОЕ ВАЖНОЕ - теперь надо выполнять следующее правило:
высота редактора на странице на 28 рх больше, чем въюмера.
То есть, когда вы делаете страницу для редактора, там, для флешки надо указать высоту (допустим) 828, а на странице для въювера 800. 
Если это правило по высоте не выполнять, будут глюки, перекосы и прочая кака. То есть, теперь для нас редактор должен быть на 28 рх выше въювера обязательно!
